How to Play:

Speelt als normaal schaken, Promoting, Rockade en En Pasant is uit
Drag & Drop je stukken om ze te verplaatsen

De tegenstander gebruikt telkens de best mogelijke zet, dus je moet gebruik maken van je Spells als je een kans wilt maken

Elke 4 beurten krijg je willekeurig een van 11 Spells, click op een spel om hem te selecteren, en vervolgens op een plaats waar je hem wilt gebruiken
Een spell gebruiken kost een beurt, en 1 maalig te gebruiken.

Om op nieuw te proberen gebruik F5

Disclaimer:

Om bepaalde tijden kan de API soms vast lopen, en dat je niet na de eerste beurt kan spelen, meestal is dit een half uur later wel weer gefixed
Bepaalde zetten kunnen ervoor zorgen dat de API vast loopt (bvb je koning schaak zetten)
